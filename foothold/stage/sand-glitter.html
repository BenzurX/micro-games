Sand Glitter - Shoreline Speckle Options

Four subtle treatments for "shining sand" on dry, unclaimed shoal tiles (SAND_COLOR #3d3018). Each tile is shown at 160×160 for readability; the real board renders these much smaller, so keep density/contrast in mind. One tile per row shows the tide-warning ring (cyan) so you can judge how glitter reads underneath it.

A - Static scatter
B - Twinkle pulse
C - Sparse glints
D - Grain streaks
A - Static scatter. A fixed field of tiny white dots at low opacity, generated once per tile and never animated. Cheapest possible (zero draw calls after the first paint, no tween overhead), reads as "sand has some sparkle in it" without drawing the eye. Risk: static specks can start to look like dust/noise rather than shine once you've stared at the board a while.
Sand fill, unchanged
~40 dots/tile, opacity 0.12-0.35, 1-2px radius
No animation - pure CSS/canvas paint, reused every refresh() via a cached texture.
B - Twinkle pulse. Same scattered dot field as A, but each speck fades in/out on its own staggered cycle (2-4s, random phase) so only a handful are "lit" at any instant. Reads as gentle shimmer, like sun catching wet grains. Costs a tween per speck (or one shared shader-style time uniform) - fine at board scale (a few dozen shoal tiles max), needs care if the board ever gets much bigger.
Same dot field as A
Each dot: opacity 0 → 0.5 → 0, 2.5-4s duration, random delay
In Phaser: a tween per dot, or one Graphics redraw per frame reading a shared sine clock (matches the existing river-flow / foam-pulse pattern already in GameScene.js).
C - Sparse glints. Far fewer points (4-6/tile) but each is a soft radial-gradient glow instead of a hard dot, occasionally flaring brighter. Reads as "a few grains catching direct light" rather than an even dusting - more restrained than A/B, closer to a jewel-like sparkle than a texture. Best if you want glitter to feel like an accent, not a base texture change.
4-6 soft glow points/tile, 3-5px radius
Each glint: base opacity 0.2, flares to 0.6 for ~300ms every 3-6s (independent random timers)
D - Grain streaks. Instead of round dots, tiny 1px-wide diagonal light streaks (like individual sand grains catching a low sun angle), all sharing one direction for a coherent "light source." No animation needed - the directionality alone reads as shine without motion. Cheapest of the four, and the most distinct from "just noise/dust" since the streaks are legible as light glinting rather than random specks.
~25 streaks/tile, 1×4px, all at ~35° angle
Opacity 0.15-0.4, varied per streak; no animation (or a very slow 8-10s drift if you want a hint of motion)
