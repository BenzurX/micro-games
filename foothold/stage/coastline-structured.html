Coastline shape — 4 structured versions

Round 3. Every ocean tile in every version below either sits on the board's outer edge or connects directly back to it, minimum 3 tiles per shoreline, and each version follows a specific, repeatable rule for how the tiles are chosen - not a pure random walk. You picked Corner Wrap as closest - it now has 1-3 inland bump tiles (outlined gold) one step in from the edge run, connected directly to it. No ocean tile in any version is allowed within 2 tiles of a home base (Chebyshev distance) anymore, so once the tidal fringe floods - 1 tile further in than the permanent ocean behind it - there's still guaranteed at least 1 dry tile of buffer before it ever reaches home. Reroll card 1 a few times to see the range; versions 2-4 are kept as-is for reference.

Shimmer treatment is the decided Layered Swell. Fringe/tide behavior (which of these tiles float/drain vs. stay permanent) isn't shown here - this page is purely about the shape question. Nothing here is built into GameScene.js yet.

1 - Corner Wrap + Bump

The same edge-hugging run as before, plus 1-2 small inland bumps (gold outline): a single tile one step in from a random point on the run - so the shoreline pokes inward in a couple spots instead of staying a perfectly flat 1-tile-wide strip, while still growing directly out of the edge run.

2 - Broken Run

Same corner-wrapping edge path as version 1, but split into 2-3 separate short runs with a gap of dry land between them - an archipelago-style broken coastline instead of one unbroken stretch.

Tiles: -

3 - L-Bracket

Always includes the exact corner tile, then a random-length arm running out along each of the two adjacent edges (arms are independently sized, so the bracket is lopsided - long one way, short the other - not a symmetric right angle).

Tiles: -

4 - Single-Edge Run

Picks just ONE of the two adjacent edges (not both) and runs a contiguous stretch of tiles along only that edge, positioned anywhere within the region - it may not touch the corner at all. Matches "sometimes the shore comes from one side altogether."
