Clicking an unlocked level selects it (gold border); Play enables once one is selected.

Design notes

Vertical list over a grid so it stays readable at phone width and the description sentence has room to breathe — a grid of square tiles would force the description into a tooltip or a second tap, which is worse for a one-time pre-match choice.

Locked/"Coming soon" levels are shown rather than hidden so the roadmap reads honestly — players can see there's more coming without it looking unfinished (no placeholder text, just a real one-line pitch and a lock icon).

Selecting a level highlights its card with a gold border (matches the game's existing gold accent for "your" state) and enables the Play button — mirrors the pattern of picking before committing, same spirit as the tile tooltip-before-commit flow in the main game.

Not decided yet: whether this screen replaces the current single "Start" button on TitleScene outright, or only appears once a second level ships (v1 stays a one-tap Start into River until there's an actual choice to make).
