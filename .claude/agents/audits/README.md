# Agent Audit Reports

One report per shipped game, produced by the process in `../AUDIT.md`.

Naming: `game-<N>-audit.md` (e.g. `game-1-audit.md`).

These are the paper trail for how the sub-agent roster evolves across the 10-game run:
what worked, what changed, and why. Read the most recent one before starting a new
game so improvements actually carry forward.
